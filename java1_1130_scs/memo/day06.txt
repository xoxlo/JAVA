제어문
프로그램의 흐름을 제어하는 것을 의미

조건문 : 조건이 있을 때 참과 거짓의 결과로 프로그램의 흐름을 바꾸는 것
반복문 : 프로그램에서 몇 번을 반복해야할 때 사용하는 것
기타제어문 : 강제로 프로그램의 흐름을 종료하거나 아래 코드를 실행하지 않을 때 사용하는 것

조건문(if문)

if (조건식1) {
	조건식1이 참일 때 실행;
}else if(조건식2) {
	조건식1이 거짓이고 조건식2가 참일 때 실행;
}else if(조건식3) {
	조건식1 거짓, 조건식2 거짓, 조건식3 참일 때 실행;
}else {
	위의 조건식이 모두 거짓일 때 실행;
}

if문 : 조건식 true면 영역({})안의 코드가 실행
else if문 : 위의 조건식이 거짓이고 else if의 조건식이 참이면 실행
else 문 : 위의 조건식들이 모두 거짓이면 실행
* else if문과 else문은 if문 없이는 사용 불가하다 !

조건문(switch문)

switch(변수명){

case 값1 :
	실행할문장;
	break; // 중괄호 영역 탈출
case 값2 :
	실행할문장;
	break;
....
default : // else와 비슷한 기능
	실행할 문장;
}

switch문에서 사용가능한 변수의 데이터 타입
byte, short, int, char, String(String JDK7부터 가능)

삼항연산자, if, switch문의 비교

삼항연산자 : 조건식 ? 참 : 거짓 -> 조건식 1개 있을 경우 사용
if문 : 조건식에 관계연산자(>, <, ..)를 사용하거나 여러개의 조건식을
	논리연산자(&&, ||, ...)로 연결할 때 사용
switch문 : 하나의 변수에 있는 값을 case의 특정 값과 같은지 비교할 때 사용

반복문(for문)

for문의 구조
for(초기식; 조건식; 증감식){
	반복할 실행문;
}

초기식 : 처음에 설정할 값
조건식 : 조건식이 true면 영역({})안의 문장 실행시키고 false for문 탈출
증감식 : 값을 얼마나 증감할지 정하는 식 ++, --, +=, -=
