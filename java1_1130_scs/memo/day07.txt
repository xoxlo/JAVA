제어문

반복문(for문)
for(초기식; 조건식; 증감식){
	반복실행할 문장;
}

초기식 : 처음에 설정할 값
조건식 : 조건이 true면 영역안의 문장 실행시키고 false면 for문 탈출
증감식 : 반복할 때 마다 값을 얼마나 증감할지 정하는 식


반복문(while문)
while(조건식){
	
}

while문과 for문의 차이
for문 : 몇번 반복할지 알때(횟수, 범위)

while문 : 몇번 반복할지 모를때


반복문(do~while문)

do{
	반복실행할 문장;
}while(조건식);

while문과 동일하지만 최소 한번은 무조건 실행해야할 때 사용한다
반복실행한 문장 먼저 실행하고 그다음 조건식 판별


이중 for문
:for문 안에 for문이 있는 구조
for(초기식; 조건식; 증감식){
	for(초기식; 조건식; 증감식){

	}
}

기타 제어문
반복문에서 사용한다. 단, break문은 switch문에서도 사용가능하다.
break : 즉시 해당 중괄호 영역을 탈추한다.
	- if문은 적용되지 않는다(if문 안에서 사용시 if문을 감싸고 있는 중괄호 탈출)
	- 아래 코드를 실행시키지 않는다
continue : 즉시 다음 반복으로 넘어간다
	- 아래 코드를 실행시키지 않는다
