2.
기본 자료형 - 정수형, 실수형, 문자형, 논리형
정수형 - short, int, long, byte
실수형 - float, double
문자형 - char
논리형 - boolean
참조 자료형 - 문자열, 배열, 클래스

기본 자료형은 stack 영역에 값 그대로 올라간다
참조 자료형은 stack 영역에 주소값이 올라간다
그 주소에는 heap영역에 있는 실제 값이 들어있는 주소이다.






1. 접근 제한자
	변수, 메소드, 클래스의 접근 권한을 제어하는 것
public : 같은 프로젝트 내에서 접근 가능
default : 다른 패키지에서 접근 불가
(별도로 명시하지 않고, 접근 권한제어자 생략시 자동으로 default)
protected : 다른 패키지에서 접근 불가, 상속 받으면 접근 가능
private : 다른 클래스에서 사용 불가, 같은 클래스만 접근 가능
	-> 메소드로만 접근 가능(getter, setter)
	사용자에게 내부 원리는 감추고 사용법만 외부로 노출시키는 것
	(캡슐화, 은닉화)
