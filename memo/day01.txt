2. 프로그램의 동작 원리
<일반 프로그램>	<JAVA프로그램 기준>
   프로그램	    프로그램
		     JVM
   OS(운영체제)	   OS(운영체제)
    하드웨어	    하드웨어

=> 자바 프로그램의 경우
프로그램 -> JVM이 프로그램 운영체제 사이에 존재
-> JVM이 운영체제와 소통하여 환경맞게 설정 -> 운영체제에 의존적이지 않다(이식성 높다)

자바의 기본 구조
1) JVM(Java Virtual Machine)
: 운영체제에 맞춰서 java로 만든 프로그램을 실행시켜준다
javac : .java(사람의 언어로 된 파일) 컴파일
java  : .class(컴퓨터의 언어로 된 파일) 실행

2) JRE(Java Runtime Environment)
: JVM을 실행한다
JVM을 실행할 때 필요한 파일 등을 가지고 있다

3) JDK(Java Development Kit)
: JRE을 포함한다

4) IDE(통합 개발 환경)
: 개발의 편의성을 높이기 위해서 사용하는 도구

- 이클립스, Visual Studio Code, 인텔리제이,...
- 이클립스 : 무료이다. 다양한 단축키를 제공하여 개발의 편의성을 높여준다

3. 이클립스 실행 후 설정

1) 이클립스 글씨체 변경

window > Preferences > font 검색 > Colors and Fonts 클릭 > Basic
> Text Font > Edit 설정 Consolas > Apply

2) 인코딩 타입 변경

window > Preferences > enc 검색 > Workspace 하단의 Text file encoding UTF-8 > Apply

3) 컴파일러 변경

window > Preferences > compiler 검색 >  JDK Compliance에서 level 변경
(기본 18 -> 17로 변경) > Apply

5) 이클립스 우측 상단의 창문 모양 클릭 > JAVA 선택 > Open
